## Running the Project in Development Mode

```bash
yarn install
yarn start
```

Open the site at [http://localhost:3000](http://localhost:3000)

## Building and Previewing the Site

```bash
yarn build
yarn serve
```

Open the site at [http://localhost:5000](http://localhost:5000)
